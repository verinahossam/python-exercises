# Task 01 - Instant Discount System

## Description

This program calculates the final purchase price after applying a discount based on the purchase amount.

## Discount Rules

- Less than 100 EGP → 0%
- 100 to less than 500 EGP → 10%
- 500 EGP or more → 20%

## Concepts Used

- Variables
- User Input
- if / elif / else
- Arithmetic Operations
- f-Strings
