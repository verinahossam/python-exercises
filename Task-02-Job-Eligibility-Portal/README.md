# Task 02 - Job Eligibility Portal

## Description

This Python program checks whether an applicant is eligible for the next interview stage based on predefined job requirements.

## Requirements

- Applicant must know Python.
- Applicant must have at least 2 years of experience or projects, OR
- Hold a Computer Science degree or Bootcamp certificate.

## Concepts Used

- Variables
- Input / Output
- String Methods
- Logical Operators
- Boolean Expressions
- if / else
